# Data Documentation

## Directory Structure
- `raw/`: Contains the raw source code files cloned from GitHub.
- `processed/`: Contains processed datasets (CSV, JSONL) ready for model training.

## Data Collection
Data is collected using the `src/data_collector.py` script.
- **Source**: GitHub API
- **Criteria**: Python repositories, >100 stars, recent activity.
